# Northstar PM - Portfolio Project

## Overview
Enterprise-grade project management interface demonstrating modern React patterns, real-time data handling, and polished UI/UX design.

## Live URL
https://skill-deploy-x7ee1dkq31-codex-agent-deploys.vercel.app

## Key Highlights

### Technical Excellence
- **Next.js 14** with App Router and server components
- **Convex** for real-time database operations
- **WorkOS integration** for enterprise auth (with demo fallback)
- **Custom design system** built from scratch (no component library dependencies)

### UX Craft
- Navy + indigo palette with warm accents
- Smooth transitions and intentional motion
- Loading states and skeleton screens
- Responsive grid layouts (4-column → 2-column → 1-column)

### Feature Set
- Dashboard with portfolio health overview
- Kanban board with status columns
- Task list with filtering and sorting
- Project detail pages with analytics
- Auth flows (WorkOS + demo mode)
