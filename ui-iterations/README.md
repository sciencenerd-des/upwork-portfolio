## Direction Notes
- `Solar Orbit`: Warm cockpit dashboard with orbit visualization and soft glass surfaces.
- `Editorial Grid`: Newspaper-like hierarchy with strict linework and typographic emphasis.
- `Coastal Cards`: Calm, fluid card system with rounded geometry and subtle wave motion.
- `Midnight Terminal`: High-density dark ops console with command and timeline semantics.
- `Brutalist Ops`: Poster-inspired brutalist interface with heavy borders and asymmetric blocks.
